\subsection{Detailed Description}
The \hyperlink{classSendableChooser}{SendableChooser} class is a useful tool for presenting a selection of options to the \hyperlink{classSmartDashboard}{SmartDashboard}.

For instance, you may wish to be able to select between multiple autonomous modes. You can do this by putting every possible \hyperlink{classCommand}{Command} you want to run as an autonomous into a \hyperlink{classSendableChooser}{SendableChooser} and then put it into the \hyperlink{classSmartDashboard}{SmartDashboard} to have a list of options appear on the laptop. Once autonomous starts, simply ask the \hyperlink{classSendableChooser}{SendableChooser} what the selected value is.
