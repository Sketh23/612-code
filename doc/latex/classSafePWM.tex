\subsection{Detailed Description}
A safe version of the \hyperlink{classPWM}{PWM} class. It is safe because it implements the \hyperlink{classMotorSafety}{MotorSafety} interface that provides timeouts in the event that the motor value is not updated before the expiration time. This delegates the actual work to a \hyperlink{classMotorSafetyHelper}{MotorSafetyHelper} object that is used for all objects that implement \hyperlink{classMotorSafety}{MotorSafety}.
