\subsection{Detailed Description}
ADXL345 \hyperlink{classAccelerometer}{Accelerometer} on \hyperlink{classSPI}{SPI}.

This class alows access to an Analog Devices ADXL345 3-\/axis accelerometer via \hyperlink{classSPI}{SPI}. This class assumes the sensor is wired in 4-\/wire \hyperlink{classSPI}{SPI} mode.
