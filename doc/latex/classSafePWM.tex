\subsection{\-Detailed \-Description}
\-A safe version of the \hyperlink{classPWM}{\-P\-W\-M} class. \-It is safe because it implements the \hyperlink{classMotorSafety}{\-Motor\-Safety} interface that provides timeouts in the event that the motor value is not updated before the expiration time. \-This delegates the actual work to a \hyperlink{classMotorSafetyHelper}{\-Motor\-Safety\-Helper} object that is used for all objects that implement \hyperlink{classMotorSafety}{\-Motor\-Safety}.
