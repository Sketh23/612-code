\subsection{\-Detailed \-Description}
\-Alias for counter class. \-Implement the gear tooth sensor supplied by \-F\-I\-R\-S\-T. \-Currently there is no reverse sensing on the gear tooth sensor, but in future versions we might implement the necessary timing in the \-F\-P\-G\-A to sense direction.
