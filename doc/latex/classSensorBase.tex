\subsection{Detailed Description}
Base class for all sensors. Stores most recent status information as well as containing utility functions for checking channels and error processing.

Add sensor to the singleton list. Add this sensor to the list of singletons that need to be deleted when the robot program exits. Each of the sensors on this list are singletons, that is they aren't allocated directly with new, but instead are allocated by the static GetInstance method. As a result, they are never deleted when the program exits.
