\subsection{\-Detailed \-Description}
\hyperlink{classI2C}{\-I2\-C} bus interface class.

\-This class is intended to be used by sensor (and other \hyperlink{classI2C}{\-I2\-C} device) drivers. \-It probably should not be used directly.

\-It is constructed by calling \hyperlink{classDigitalModule_ae776c8449cf18b7f90fec0126b3e8373}{\-Digital\-Module\-::\-Get\-I2\-C()} on a \hyperlink{classDigitalModule}{\-Digital\-Module} object.
