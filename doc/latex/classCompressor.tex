\subsection{\-Detailed \-Description}
\hyperlink{classCompressor}{\-Compressor} object. \-The \hyperlink{classCompressor}{\-Compressor} object is designed to handle the operation of the compressor, pressure sensor and relay for a \-F\-I\-R\-S\-T robot pneumatics system. \-The \hyperlink{classCompressor}{\-Compressor} object starts a task which runs in the backround and periodically polls the pressure sensor and operates the relay that controls the compressor.
