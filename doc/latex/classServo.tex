\subsection{\-Detailed \-Description}
\-Standard hobby style servo.

\-The range parameters default to the appropriate values for the \-Hitec \-H\-S-\/322\-H\-D servo provided in the \-F\-I\-R\-S\-T \-Kit of \-Parts in 2008.
