The constructor for a \hyperlink{classMotorSafetyHelper}{MotorSafetyHelper} object. The helper object is constructed for every object that wants to implement the Motor Safety protocol. The helper object has the code to actually do the timing and call the motors Stop() method when the timeout expires. The motor object is expected to call the Feed() method whenever the motors value is updated. 
\begin{DoxyParams}{Parameters}
{\em safeObject} & a pointer to the motor object implementing \hyperlink{classMotorSafety}{MotorSafety}. This is used to call the Stop() method on the motor. \\
